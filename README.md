## Run locally

No installation needed:

```bash
git clone https://github.com/ericgitau-tech/eric-gitau-portfolio.git
cd <https://github.com/ericgitau-tech/eric-gitau-portfolio.git>
open index.html   # or just double-click the file
```

## Deploy

This site is designed to be hosted for free with **GitHub Pages**:

1. Push `index.html` to this repository (on the `main` branch)
2. Go to **Settings → Pages**
3. Under **Build and deployment → Source**, select **Deploy from a branch**
4. Choose the `main` branch and `/ (root)` folder, then **Save**
5. The site will publish at the URL shown on that page within a minute or two
